# \[Team. MINT\] 프로젝트 기획안: Cost-Vue (코스트-뷰)

"세상의 정보를 녹여, 당신의 숫자로 주조합니다."

본 기획안은 인텔 AI 과정을 통해 확보한 기술적 역량(Claude 3.5, Exa)을
바탕으로, 복잡한 글로벌 뉴스를 경제활동 인구의 실생활 지표로 변환하는
**AI 기반 인과관계 저널리즘 서비스**를 정의합니다.

------------------------------------------------------------------------

## 1. 서비스 개요 (Executive Summary)

-   **서비스명**: Cost-Vue (코스트-뷰)\
-   **슬로건**: 뉴스를 지갑의 숫자로 번역하다.\
-   **핵심 가치**: Fact-based / Causal Logic / Active Defense\
-   **타겟 고객**: 합리적 소비와 지출 관리를 원하는 경제활동 인구 전체

------------------------------------------------------------------------

## 2. 시장의 문제점 및 기회 (Problem & Opportunity)

-   **정보 과잉**: 경제 뉴스는 많지만 개인 지출 영향은 불명확\
-   **인과관계 난해성**: 거시경제 → 생활비 연결 구조 이해 어려움\
-   **저널리즘 공백**: 감정 배제된 데이터 중심 결론 부족

------------------------------------------------------------------------

## 3. 핵심 기능 (Key Features)

### \[1\] 2-Step 인과관계 도식화 (The Logic Chain)

-   사건 → 결과 구조로 단순화\
-   예: 브라질 가뭄 → 커피 가격 상승 가능성 80%\
-   화살표 기반 시각화 제공

### \[2\] 지표 신호등 (Wallet Traffic Light)

-   주요 지출 항목 위험도 표시\
-   초록 / 노랑 / 빨강 3단계

### \[3\] 팩트 제련소 (The Fact Mint)

-   Exa 기반 검증 뉴스 수집\
-   Claude 기반 편향 제거 요약

------------------------------------------------------------------------

## 4. 기술 스택 및 구현 전략 (Technical Blueprint)

  구분          기술 스택             역할
  ------------- --------------------- --------------------
  추론 엔진     Claude 3.5 Sonnet     인과관계 구조화
  검색 엔진     Exa Semantic Search   고품질 데이터 수집
  검증 로직     Cross-Verification    신뢰도 확보
  데이터 연동   Real-time Price API   가격 지표 매핑

------------------------------------------------------------------------

## 5. 서비스 로드맵 (4-Week NLP Sprint)

-   **1주차**: 시맨틱 쿼리 최적화\
-   **2주차**: 인과관계 추출 체인 구축\
-   **3주차**: Back-testing 검증\
-   **4주차**: 최종 데이터 엔진 완성

------------------------------------------------------------------------

## 6. 기대 효과 (Expected Impact)

-   정보 격차 해소\
-   선제적 소비 전략 가능\
-   데이터 저널리즘 표준 구축
